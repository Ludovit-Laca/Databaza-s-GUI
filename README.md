# Databaza-s-GUI (12-17 bodov)

Vytvorte systém s minimálne troma tabuľkami so správne navrhnutými reláciami (3 body), ktorý umožní zobrazovanie údajov nasledovným spôsobom:

- Výber riadku z prvej tabuľky definuje zobrazovanie údajov v druhej tabuľke:
  - buď v rovnakom alebo samostatnom okne (1 bod)
  - v MDI okne so všetkými vymoženosťami, ktoré MDI poskytujú (3 body)
  - kombinácia údajov z viacerých tabuliek vo východiskovej tabuľke (2 body)
- Pridávanie, editácia a mazanie so všetkými kontrolami nad oboma tabuľkami (4 body)
- Filtrovanie nad každým stĺpcom prvej tabuľky okrem ID - vyžaduje sa nejaká univerzálna metóda (metódy) (2 body)
- Vyhľadávanie v tabuľke v zmysle hľadania zadaného reťazca vo vybranom alebo vo všetkých poliach a nastavením kurzora na riadok, ktorý spĺňa podmienku (2 body)
